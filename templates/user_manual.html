4. 系统登录

4.1 登录流程

  1. 打开浏览器（建议使用 Chrome / Edge 最新版），在地址栏输入系统访问地址：http://192.168.1.25:8080/login（其中 192.168.1.25 须替换为部署本系统的服务器本机内网 IP 地址，端口号 8080 替换为运维实际配置的服务端口；本机 IP 可登录服务器执行 ipconfig（Windows）或 ifconfig / ip addr（Linux）查询）
  2. 进入登录页面后，在"账户登录"区域输入您分配到的用户名与密码。
  3. 如首次登录，请使用管理员下发的初始账号（默认用户名 admin，密码 123456），登录后请立即修改密码。
  4. 可勾选"记住登录状态"以便短期内免登录（公共电脑上建议不勾选）。
  5. 点击"登 录 账 户"按钮，系统校验成功后自动跳转至工作台首页。
